## Algorithm supported
| Algorithm         | Deno  | Node.js |
| ----------------- | :---: | :-----: |
| `Ed25519`         |   ✔   |    ✔    |
| `P-256` / `ES256` |   ✔   |    ✔    |
| `P-384` / `ES384` |   ✔   |    ✔    |
| `P-521` / `ES512` |       |    ✔    |
